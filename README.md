## Project Description
A collection of terminal commands, shell scripts and Git workflow exercises 
built during Week 3 of the IYF Weekend Academy Season 11. The goal was to 
learn how to navigate the terminal, automate tasks using shell scripts, and 
use Git professionally.

## Technologies Used
- Bash scripting
- Git & GitHub
- Command Line Interface (CLI)
- Markdown

## Features
- Project setup script that automatically creates a folder structure
- HTML boilerplate generator script
- Professional `.gitignore` file
- Git branching and merging workflow

## How to Run
1. Clone this repository:
git clone https://github.com/helter-skelter254/iyf-s11-week-03-helter-skelter254.git

2. Make scripts executable:
chmod +x new-project.sh
chmod +x boilerplate.sh

3. Run the project setup script:
./new-project.sh my-project-name

## Lessons Learned
- How to navigate the file system using only the terminal
- How to create, copy, move and delete files using CLI commands
- How to write bash scripts to automate repetitive tasks
- How to use Git branching to work safely without affecting the main codebase
- How to use .gitignore to keep unnecessary files out of GitHub

## Challenges Faced
- Understanding the difference between git pull and git fetch
- Resolving merge conflicts when two branches have different changes
- Learning bash script syntax which is different from HTML and CSS
